## 正则表达式
同时包含数字和字母，6-10位：`/^(?![0-9]+$)(?![a-zA-Z]+$)[0-9A-Za-z]{6,10}$/`

- ^ 匹配一行的开头位置
- (?![0-9]+$) 预测该位置后面不全是数字
- (?![a-zA-Z]+$) 预测该位置后面不全是字母
- [0-9A-Za-z] {6,10} 由6-10位数字或这字母组成
- $ 匹配行结尾位置
